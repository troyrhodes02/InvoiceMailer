# InvoiceMailer User Guide - Windows Version

Welcome to InvoiceMailer!

This app lets you send invoice emails automatically using your Microsoft 365 tenant.

---

## How to Set Up:

1. Extract the InvoiceMailer zip file to a folder on your computer.
2. Double-click `InvoiceMailerUI.exe` to start the application.
3. On first launch, enter your:
   - TenantId
   - ClientId
   - ClientSecret
   (The app will save these automatically.)

4. Edit `recipients.csv` to map invoice keys to email addresses.
5. Place invoice files into the `invoices` folder.

---

## How to Run:

Every time you want to send invoices:
- Double-click `InvoiceMailerUI.exe`

---

## Troubleshooting:

If you get a security warning:
1. Right-click `InvoiceMailerUI.exe`
2. Select "Properties" 
3. Check "Unblock" option
4. Click "Apply" and "OK"
